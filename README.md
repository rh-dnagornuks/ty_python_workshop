# Installation

1. Clone the repo and change current directory to it.
```
git clone https://github.com/rh-dnagornuks/ty_python_workshop.git
cd ty_python_workshop
```

2. Create virtual Python environment, activate it, install 'jupyter-lab' package and run it.
```
python -m venv venv
source venv/bin/activate
pip install jupyterlab
jupyter-lab
```

3. Double click on the 'ty_python_workshop.ipynb' file in the directory view in the notebook.
